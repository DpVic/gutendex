# Book API Project

## Installation

1. **Build and start the containers:**
   ```bash
   make install
   ```

2. **Run tests:**
   ```bash
   make test
   ```

3. **Check code quality:**
   ```bash
   make pre-push
   ```

4. **API Documentation:**
   The documentation is available at [http://localhost:8084/](http://localhost:8084/)

For more details about the project requirements, see [INSTRUCTIONS.md](./INSTRUCTIONS.md).
